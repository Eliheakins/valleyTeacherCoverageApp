# valleyTeacherCoverageApp
SCHEDULE FORMATTING REQUIREMENTS

1. File and Column Naming
File Type: Must be an Excel file named Coverage_Schedule.xlsx 

Needed column headers: 
    These column headers must be in the first row of the excel sheet.
    a.
    Duty Column Headers: Must be formatted as "Duty [Ordinal Suffix]".
    Duty Header Examples: Duty 1st, Duty 2nd, Duty 3rd, ... Duty 11th
    b.
    Name 
    c.
    Periods: Must be formatted as [Ordinal Suffix]
    Example: 1st, 2nd, 3rd
    d.
    Periods needing covered: Must be formatted as Need Coverage

2. Teacher Naming Conventions
Teacher Name (in the "Name" column): Must be formatted as "[Last Name], [First Name]" with the last name first.
Optional: Can have the teacher a sub is long-term covering for below sub's name in parenthesis

Duty Assignment (Inside Duty columns): Any duty assignment must contain the teacher's name in following format "[Last Name], [First Name]" written the same as their name is specified in the "Name" column

3. "Need Coverage" Column Formatting
The values in the "Need Coverage" column are highly specific.

Standard Periods: Periods must be simple number strings, separated by commas. Example: 1, 4, 11

Split Periods: Periods must be joined by a single slash (/), with no spaces, and separated from other periods by a comma. Example: 5/6, 8/9

Co-Taught Periods: Must strictly follow the pattern: [Standard Periods] CT-[Co-Taught Periods]. Example: 1,4,11 CT-2,5/6,8/9,10

4. Special Codes in Duty Columns
ISS Duty: The text "ISS" must appear anywhere in the duty cell to flag that assignment as ISS coverage. Example: Fry ISS, ISS Fry
